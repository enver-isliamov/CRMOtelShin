# Секреты и Переменные Окружения (Environment Variables)

В этом файле перечислены все переменные окружения, которые используются в проекте. Они необходимы для корректной работы базы данных, Telegram-бота и автоматических задач (Cron Jobs).

> **Где они хранятся:** Все эти переменные должны быть добавлены в панель управления **Vercel** в разделе **Settings -> Environment Variables** вашего проекта.

---

## 1. База Данных (Vercel Postgres / Neon)
Эти переменные создаются автоматически при подключении базы данных Vercel Postgres к вашему проекту. Если вы используете стороннюю базу (например, Supabase или Neon напрямую), вам нужно задать их вручную.

*   `POSTGRES_URL` (или `DATABASE_URL`)
    *   **Описание:** Основная строка подключения к базе данных PostgreSQL (Connection String).
    *   **Пример:** `postgres://user:password@host:5432/dbname`
    *   **Где используется:** Во всех API-роутах (`api/crm.ts`, `api/bot.ts`, `api/setup.ts`, `api/ping.ts`, `api/cron-reminders.ts`).
*   `STOREGE_POSTGRES_URL`
    *   **Описание:** Резервное имя переменной (использовалось ранее из-за опечатки при настройке Vercel Storage). Код поддерживает его для обратной совместимости.
*   `POSTGRES_URL_NON_POOLING`
    *   **Описание:** Строка подключения без пулинга (иногда полезна для скриптов миграции).

---

## 2. Telegram Bot
Эти переменные необходимы для работы бота-ассистента и отправки уведомлений.

*   `TELEGRAM_BOT_TOKEN`
    *   **Описание:** Уникальный токен вашего Telegram-бота, полученный от [@BotFather](https://t.me/BotFather).
    *   **Пример:** `1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ`
    *   **Где используется:** `api/bot.ts` (прием сообщений), `api/crm.ts` (отправка уведомлений менеджерам и клиентам), `api/cron-reminders.ts` (отправка напоминаний).
*   `ADMIN_CHAT_ID`
    *   **Описание:** Ваш личный Telegram Chat ID (или ID группы менеджеров). Сюда бот будет присылать уведомления о новых лидах из Mini App и заявках на выдачу шин.
    *   **Пример:** `123456789`
    *   **Где используется:** `api/bot.ts` (уведомления о заявках), `api/crm.ts` (уведомления о новых лидах).

---

## 3. Автоматизация (Cron Jobs)
Эти переменные нужны для защиты автоматических скриптов от несанкционированного запуска извне.

*   `CRON_SECRET`
    *   **Описание:** Секретный ключ, который Vercel автоматически отправляет в заголовке `Authorization: Bearer ...` при вызове Cron-задач. Вы придумываете его сами.
    *   **Пример:** `my_super_secret_cron_key_12345`
    *   **Где используется:** `api/ping.ts` (защита пинга БД), `api/cron-reminders.ts` (защита рассылки напоминаний).

---

## 4. Системные переменные Vercel
Эта переменная генерируется самим Vercel автоматически.

*   `VERCEL_URL`
    *   **Описание:** Текущий домен вашего проекта на Vercel (без `https://`).
    *   **Где используется:** В `api/crm.ts` (действие `set_bot_webhook`) для автоматической привязки вебхука Telegram к вашему актуальному домену.
